# Advanced Git: achieving utopic git history

A slide deck and exercises for a Coding Cafe session on advanced git commands that help you achieve a clean "Utopian" Git history. 

This Cafe session was organized in January 2024 and contained a presentation with a slide deck, mixed with some 'live coding' examples. It is important to carefully check the background of the audience at the beginning using the "Who has used..." slide. Based on this, the presenter can tailor the live coding examples towards the commands that are not used by the majority of the audience. 

The exercises folder contains links to a variety of exercises to practice the covered git commands.

This repository is not actively maintained an may contain broken links or outdated git commands.
